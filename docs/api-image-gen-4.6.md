## 接口简介

即梦图片4.6模型——即梦同源的图像生成能力。该模型是基于Seedream4.0基础模型训练的生图版本，聚焦提升了大众修图垂类场景的效果，包含人像写真、平面设计和图片风格化。

## 接入说明

### 限制条件

| 名称 | 内容 |
|------|------|
| **输入图要求** | 1. 图片格式：仅支持JPEG、PNG格式，建议使用JPEG格式；2. 图片文件大小：最大 15MB，支持最多输入14张图；3. 图片分辨率：最大 4096 * 4096 |
| **输出图说明** | 1. 输出图详细宽高规则，参考下方width、height参数描述；2. 输出图会以列表形式返回，最大输出图数量 = 15 - 输入图数量，建议输出图数量不超过6张； |
| **其他说明** | 1. 输入/输出图片的分辨率越高、数量越多，延迟增加越明显；2. 一次调用可能输出多张图片，根据输出图片张数进行计费，默认根据prompt理解意图判断输出图片数量；3. 若对延迟&价格敏感，建议使用force_single参数，强制只输出1张图片 |

### 请求说明

| 名称 | 内容 |
|------|------|
| 接口地址 | `https://visual.volcengineapi.com` |
| 请求方式 | POST |
| Content-Type | application/json |

## 提交任务

### Query参数

拼接到url后的参数，示例：
```
https://visual.volcengineapi.com?Action=CVSync2AsyncSubmitTask&Version=2022-08-31
```

### Header参数

> **注意：** 本服务固定值：Region为`cn-north-1`，Service为`cv`。主要用于鉴权，详见 公共参数 - 签名参数 - 在Header中的场景部分。

### Body参数

> **注意：** 业务请求参数，放到request.body中，MIME-Type为 application/json

| 参数 | 类型 | 可选/必选 | 说明 |
|------|------|-----------|------|
| Action | string | 必选 | 接口名，取值：`CVSync2AsyncSubmitTask` |
| Version | string | 必选 | 版本号，取值：`2022-08-31` |
| req_key | string | 必选 | 服务标识，取固定值: `jimeng_seedream46_cvtob` |
| image_urls | array of string | 可选 | 图片文件URL，支持输入0至14张图。图片数量越多&图片尺寸越大，整体服务处理时间越长。输入图片建议控制在6张图片以内，图片过多，参考效果会降低 |
| prompt | string | 必选 | 用于生成图像的提示词，中英文输入均可。<br>**建议：** 最长不超过800字符，prompt过长有概率出图异常或不生效；支持prompt中直接指定生图比例，模型会根据"size"字段智能判断生图宽高比，默认生成2K分辨率、宽高比不超过3的图像；除""外不建议输入特殊的符号如$ |
| size | int | 可选 | **文生图场景：** 面积和宽高需要2选1传入，都不传则默认面积取 4194304（即2048*2048），生成2K分辨率图像且模型智能判断宽高比；面积和宽高同时输入时，优先使用宽高；只传面积，模型会根据用户prompt意图智能判断生图宽高比。<br>**图生图场景：** 结合用户prompt意图、参考图尺寸，生成图片的面积。<br>**默认值：** 4194304，即2048*2048，生成2K分辨率图像；**取值范围：** [1024*1024, 4096*4096]，可生成1K到4K分辨率图像 |
| width | int | 可选 | 生成图像宽高，默认该字段不传；宽高乘积在 [1024*1024, 4096*4096]，且宽高比在[min_ratio, max_ratio]之间。**注意：** 需同时传width和height才会生效 |
| height | int | 可选 | 由模型智能判断生图宽高比 |
| scale | int | 可选 | 文本描述影响的程度，该值越大代表文本描述影响程度越大，且输入图片影响程度越小。**默认值：** 50；**取值范围：** [1, 100] |
| force_single | bool | 可选 | 是否强制生成单图。**默认值：** false。生成的组图越多，耗时越久，且生成耗时会随图片数量增多而显著变长；如需稳定的组图输出效果，建议prompt控制组图数量不超过6张 |
| min_ratio | float | 可选 | 生图结果的宽/高 ≥ min_ratio。**默认值：** 1/3；**取值范围：** [1/16, 16) |
| max_ratio | float | 可选 | 生图结果的宽/高 ≤ max_ratio。**默认值：** 3；**取值范围：** (1/16, 16] |

### 推荐可选的宽高

**1K：** 1024x1024 (1:1)

**2K：**
- 2048x2048 (1:1)
- 2304x1728 (4:3)
- 2496x1664 (3:2)
- 2560x1440 (16:9)
- 3024x1296 (21:9)

**4K：**
- 4096x4096 (1:1)
- 4693x3520 (4:3)
- 4992x3328 (3:2)
- 5404x3040 (16:9)
- 6197x2656 (21:9)

### 异步回调参数说明

> 下述参数仅对异步回调生效，使用异步轮询调用方式时无需配置

| 参数 | 类型 | 可选/必选 | 说明 |
|------|------|-----------|------|
| callback_url | string | 可选 | 回调接口URL（需公网可访问） |
| return_url | bool | 可选 | 在使用异步回调时，设置图片资源是否以链接形式返回（链接有效期为24小时）。**默认值：** false |
| logo_info | JSON string | 可选 | 水印信息（JSON序列化字符串） |
| aigc_meta | JSON string | 可选 | 隐式标识（JSON序列化字符串） |

#### LogoInfo 水印配置

| 名称 | 类型 | 可选/必选 | 描述 |
|------|------|-----------|------|
| add_logo | bool | 可选 | 是否添加水印。**默认值：** false |
| position | int | 可选 | 水印位置：0-右下角、1-左下角、2-左上角、3-右上角。**默认值：** 0 |
| language | int | 可选 | 水印语言：0-中文（AI生成）、1-英文（Generated by AI）。**默认值：** 0 |
| opacity | float | 可选 | 不透明度，取值范围0-1。**默认值：** 1 |
| logo_text_content | string | 可选 | 明水印自定义内容 |

> **传参示例：** `"logo_info": "{\"add_logo\":true, \"position\": 0,\"language\": 0,\"opacity\": 1,\"logo_text_content\":\"这里是明水印内容\"}"`

#### AIGCMeta 隐式标识

| 名称 | 类型 | 可选/必选 | 描述 |
|------|------|-----------|------|
| content_producer | String | 可选 | 内容生成服务ID |
| producer_id | String | **必选** | 内容生成服务商给此图片数据的唯一ID |
| content_propagator | String | 可选 | 内容传播服务商ID |
| propagate_id | String | 可选 | 传播服务商给此图片数据的唯一ID |

> **传参示例：** `"aigc_meta": "{\"content_producer\": \"xxxxxx\", \"producer_id\":\"xxxxxx\", \"content_propagator\": \"xxxxxx\", \"propagate_id\":\"xxxxxx\"}"`

### 提交任务返回参数

| 字段 | 类型 | 说明 |
|------|------|------|
| task_id | string | 任务ID，用于查询结果 |

### 提交任务请求 & 返回完整示例

**请求示例：**
```json
{
  "req_key": "jimeng_seedream46_cvtob",
  "image_urls": ["https://xxxx"],
  "prompt": "千军万马"
}
```

**返回示例：**
```json
{
  "code": 10000,
  "data": {
    "task_id": "7392616336519610409"
  },
  "message": "Success",
  "request_id": "20240720103939AF0029465CF6A74E51EC",
  "time_elapsed": "104.852309ms"
}
```

> **状态码说明：** code=10000 表示成功，code!=10000 的情况下，不会返回task_id。

## 查询任务

### Query参数

```
https://visual.volcengineapi.com?Action=CVSync2AsyncGetResult&Version=2022-08-31
```

| 参数 | 类型 | 可选/必选 | 说明 |
|------|------|-----------|------|
| Action | string | 必选 | 固定值：`CVSync2AsyncGetResult` |
| Version | string | 必选 | 固定值：`2022-08-31` |

### Body参数

| 参数 | 类型 | 可选/必选 | 说明 |
|------|------|-----------|------|
| req_key | string | 必选 | 固定值: `jimeng_seedream46_cvtob` |
| task_id | string | 必选 | 任务ID |
| req_json | JSON string | 可选 | json序列化后的字符串。支持return_url、logo_info、aigc_meta |

> **传参示例：** `"req_json": "{\"return_url\": true}"`

### 查询任务返回参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| binary_data_base64 | array of string | 返回图片的base64数组 |
| image_urls | array of string | 返回图片的url数组，输出图片格式为png格式（有效期是 24h） |
| status | string | 任务执行状态：`in_queue`(已提交)、`generating`(处理中)、`done`(完成)、`not_found`(未找到/过期)、`expired`(已过期，请重新提交) |

### 查询任务返回示例

**成功：**
```json
{
  "code": 10000,
  "data": {
    "binary_data_base64": null,
    "image_urls": ["https://xxxx"],
    "status": "done"
  },
  "message": "Success",
  "status": 10000,
  "request_id": "2025061718460554C9B78D23B0BAB45B2A",
  "time_elapsed": "508.312154ms"
}
```

> **注意：** 状态码优先判断 code=10000，然后再判断data.status。

**失败：**
```json
{
  "code": 50413,
  "data": null,
  "message": "Post Text Risk Not Pass",
  "request_id": "202511281418218670D408837A9B0EB58F",
  "status": 50413,
  "time_elapsed": "36.799829ms"
}
```

## 错误码

### 业务错误码

| HttpCode | 错误码 | 错误消息 | 描述 | 重试 |
|----------|--------|----------|------|:----:|
| 200 | 10000 | 无 | 请求成功 | 否 |
| 400 | 50411 | Pre Img Risk Not Pass | 输入图片前审核未通过 | 否 |
| 400 | 50511 | Post Img Risk Not Pass | 输出图片后审核未通过 | 可 |
| 400 | 50412 | Text Risk Not Pass | 输入文本前审核未通过 | 否 |
| 400 | 50512 | Post Text Risk Not Pass | 输出文本后审核未通过 | 否 |
| 400 | 50413 | Post Text Risk Not Pass | 输入文本含敏感词、版权词 | 否 |
| 400 | 50518 | Pre Img Risk Not Pass: Copyright | 输入版权图前审核未通过 | 否 |
| 400 | 50519 | Post Img Risk Not Pass: Copyright | 输出版权图后审核未通过 | 可 |
| 400 | 50520 | Risk Internal Error | 审核服务异常 | 否 |
| 400 | 50521 | Antidirt Internal Error | 版权词服务异常 | 否 |
| 400 | 50522 | Image Copyright Internal Error | 版权图服务异常 | 否 |
| 429 | 50429 | Request Has Reached API Limit | QPS超限 | 可 |
| 429 | 50430 | Request Has Reached API Concurrent Limit | 并发超限 | 可 |
| 500 | 50500 | Internal Error | 内部错误 | 否 |
| 500 | 50501 | Internal RPC Error | 内部算法错误 | 否 |
