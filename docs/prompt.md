# Prompt 库

> 所有图片生成、视频生成的 prompt 统一维护在此文档。
> 如需更新 prompt 策略，只修改本文档。
> **生成工具**：即梦 API（图片 I2I + 视频 I2V 首尾帧模式）

---

## 一、图片生成 Prompt

### Step 1-A｜角色标准正面图

**Case 1 — 风格不符**（用户选择风格与上传图不一致）
- **生成方式**：即梦 I2I，上传原图为附件
- **Prompt**：`根据附件图的人物角色，重新生成一张[用户所选风格]的角色正面图，人物保持不变`

---

**Case 2 — 风格一致但图片不达标**（分三种子情况，均上传原图为附件）

| 子情况 | 触发条件 | Prompt |
|--------|----------|--------|
| 2-A | 清晰度不达标（<1024×1024），构图达标 | `保持原图不变，修正清晰度为1024×1024以上` |
| 2-B | 清晰度达标，构图不达标（非正面/微侧面） | `保持附件图人物、表情、姿势、背景一致，人物变更为正面图` |
| 2-C | 清晰度与构图均不达标 | `保持附件图人物、表情、姿势、背景一致，人物变更为正面图，并修正清晰度为1024×1024以上` |

### Step 1-A｜中性帧

- **生成方式**：即梦 I2I，上传角色标准正面图为附件
- **触发时机**：角色标准正面图生成后自动生成
- **Prompt**：`保持镜头固定不变，保持人物一致，角色看向镜头，仿佛在聆听一般`

---

### Step 1-C｜场景结合图（角色在场景中）

- **生成方式**：即梦 I2I，上传角色标准正面图 + 场景背景图作为附件
- **Prompt**：待细化（需体现角色在场景中做符合人设的事情）

---

### Step 1-C｜对话半身特写图

- **生成方式**：即梦 I2I，基于场景结合图
- **Prompt**：待细化（角色看向镜头，背景与场景一致，半身构图）

---

## 二、视频生成 Prompt

### Step 1-C｜5s 预览视频

- **生成方式**：即梦 I2V，首尾帧 = 角色标准正面图
- **时长**：5s
- **Prompt**：`保持镜头不变，保持人物的一致性，角色有轻微呼吸起伏，持续做着自己的事情`

---

### 固定前缀（长待机 / 小动作1 / 小动作2 / 说话 / 聆听）

`固定镜头，镜头和背景都不做任何变化，保持人物一致性，`

---

### 长待机（10s）

`固定镜头，镜头和背景都不做任何变化，保持人物一致性，` + 【状态1描述中提取1句】 + `角色动作轻微起伏，专注自己的事情上。`

### 小动作1（10s）

- **首尾帧**：角色标准正面图
- **生成逻辑**：基于角色定稿图的图片内容做客观描述，动作幅度极小
- **Prompt 结构**：固定前缀 + 【身体姿态 + 具体动作 + 表情/眼神 + 环境氛围，不看镜头】

```
固定镜头，镜头和背景都不做任何变化，保持人物一致性，{身体姿态}，{具体动作}，{表情/眼神}，{环境氛围}，不看镜头。
```

> 括号内容由 agent 基于**角色定稿图的图片描述**直接生成，描述图中客观呈现的状态，不做推断。

---

### 小动作2（10s）

- **首尾帧**：角色标准正面图
- **生成逻辑**：基于角色档案.txt 中的性格/兴趣爱好，推测角色在此时可能产生的一个微小自然反应，动作幅度极小
- **Prompt 结构**：固定前缀 + 【身体姿态 + 具体动作 + 表情/眼神 + 环境氛围，不看镜头】

```
固定镜头，镜头和背景都不做任何变化，保持人物一致性，{身体姿态}，{具体动作（符合人设的微变）}，{表情/眼神}，{环境氛围}，不看镜头。
```

> 括号内容由 agent 基于**角色档案.txt 的性格/兴趣爱好字段**推断生成，体现角色人设下自然产生的细微行为，非客观描述。

### 说话（3s）

- **首尾帧**：中性帧
- **生成逻辑**：基于角色档案.txt 中的性格/标志特征，推测角色开口说话时的姿态与表情
- **Prompt 结构**：固定前缀 + 【身体姿态 + 具体动作 + 表情/眼神，看向镜头】

```
固定镜头，镜头和背景都不做任何变化，保持人物一致性，{身体姿态}，{具体动作}，{表情/眼神（符合人设的说话状态）}，看向镜头。
```

> 括号内容由 agent 基于**角色档案.txt 的性格/标志特征字段**推断生成。

---

### 聆听（3s）

- **首尾帧**：中性帧
- **生成逻辑**：基于角色档案.txt 中的性格/标志特征，推测角色聆听时的姿态与表情
- **Prompt 结构**：固定前缀 + 【身体姿态 + 具体动作 + 表情/眼神，看向镜头】

```
固定镜头，镜头和背景都不做任何变化，保持人物一致性，{身体姿态}，{具体动作}，{表情/眼神（符合人设的聆听状态）}，看向镜头。
```

> 括号内容由 agent 基于**角色档案.txt 的性格/标志特征字段**推断生成。

### 过渡1 / transition_state_change（3s）

- **首帧**：角色标准正面图
- **尾帧**：中性帧
- **Prompt**：`保持镜头固定不变，保持人物一致，角色看向镜头，仿佛在聆听一般`

### 过渡2 / transition_return_to_idle（3s）

- **首帧**：中性帧
- **尾帧**：角色标准正面图
- **Prompt**：`固定镜头，镜头和背景都不做任何变化，保持人物一致性，角色视线从镜头移开，{身体姿态}，{具体动作}，{表情/眼神}，不看镜头。`

> 括号内容由 agent 基于**角色档案.txt 的兴趣爱好/标志特征字段**推断生成，描述角色自然回到自己日常状态的动作。
