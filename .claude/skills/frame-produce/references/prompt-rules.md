# 视频 Prompt 规则

固定前缀（长待机/小动作/说话/聆听）：
`固定镜头，镜头和背景都不做任何变化，保持人物一致性，`

## 图片标准化 Prompt

| 情况 | Prompt |
|------|--------|
| 风格不符（Case 1） | `根据附件图的人物角色，重新生成一张{风格}的角色正面图，人物保持不变` |
| 清晰度不达标（Case 2-A） | `保持原图不变，修正清晰度为1024×1024以上` |
| 构图不达标（Case 2-B） | `保持附件图人物、表情、姿势、背景一致，人物变更为正面图` |
| 两者均不达标（Case 2-C） | `保持附件图人物、表情、姿势、背景一致，人物变更为正面图，并修正清晰度为1024×1024以上` |
| 完全达标 | 不调用 I2I，直接使用原图 |

中性帧固定 Prompt：`保持镜头固定不变，保持人物一致，角色看向镜头，仿佛在聆听一般`

## 7 个视频 Prompt 规则

| 视频 | 生成方式 | Prompt 结构 |
|------|----------|-------------|
| idle_001（长待机） | 固定 | 固定前缀 + 状态1描述中提取1句 + `角色动作轻微起伏，专注自己的事情上。` |
| idle_002（小动作1） | Claude 生成 | 固定前缀 + 基于**角色定稿图客观描述**：`{身体姿态}，{具体动作}，{表情/眼神}，{环境氛围}，不看镜头。` |
| idle_003（小动作2） | Claude 生成 | 固定前缀 + 基于**档案性格/兴趣爱好推断微变**：`{身体姿态}，{具体动作（符合人设的微变）}，{表情/眼神}，{环境氛围}，不看镜头。` |
| transition_state_change_001 | 固定 | `保持镜头固定不变，保持人物一致，角色看向镜头，仿佛在聆听一般` |
| transition_return_to_idle_001 | Claude 生成 | 固定前缀 + 基于**档案兴趣爱好/标志特征**推断回到自己日常状态：`角色视线从镜头移开，{身体姿态}，{具体动作}，{表情/眼神}，不看镜头。` |
| listen_001（聆听） | Claude 生成 | 固定前缀 + 基于**档案性格/标志特征**推断聆听状态：`{身体姿态}，{具体动作}，{表情/眼神（符合人设的聆听状态）}，看向镜头。` |
| dialogue_base_001（说话） | Claude 生成 | 固定前缀 + 基于**档案性格/标志特征**推断说话状态：`{身体姿态}，{具体动作}，{表情/眼神（符合人设的说话状态）}，看向镜头。` |

> idle_002 描述图中**客观呈现**的状态，不做推断。idle_003 体现角色人设下自然产生的**细微行为**，非客观描述。
