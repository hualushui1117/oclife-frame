# 状态描述规则

基于角色档案 + 已确定的场景/事件，生成三段状态描述。

## 状态1 — 沉浸

**句数**：3-5 句  
**视角**：不看镜头  
**结构**：身体姿态 + 具体动作 + 表情/眼神 + 环境氛围

## 状态2 — 微变

**句数**：2-3 句  
**规则**：在状态1基础上叠加一个符合角色人设的轻微变化，动作幅度极小

**微变方向参考（按事件类型）**：

| 事件类型 | 微变方向 |
|----------|----------|
| 睡觉/休息 | 轻轻翻身、蹬腿、耳朵动、嘴巴咂一下、调整睡姿 |
| 看书/写字 | 翻一页、调整坐姿、揉眼睛、托腮、轻叹 |
| 画画 | 换笔、歪头看画布、退远端详、调色 |
| 发呆 | 换方向看、轻叹、托腮、抱膝、低头 |
| 玩耍/摆弄物品 | 换一个物品、重新排列、歪头端详、拿起放下 |

## 状态3 — 发现

**句数**：3-4 句  
**结构**：察觉 → 确认 → 回应，视线转向镜头

## 保存路径

```
session/{角色名}的场景事件状态描述.txt
```

## 文件格式

```
场景：{场景名称及描述}
事件：{角色正在这个场景里持续做什么}

【状态1 - 沉浸】
{3-5句描述}

【状态2 - 微变】
{2-3句描述}

【状态3 - 发现】
{3-4句描述}
```
